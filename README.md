# 🏠 WanderLust

A full-stack Airbnb-inspired property listing web application built using **Node.js, Express.js, MongoDB, EJS, and Bootstrap**.

---

## ✨ Features

- 📝 Add New Listings
- ✏️ Edit Existing Listings
- 🗑️ Delete Listings
- ⭐ Add Reviews
- 🖼️ Default Property Images
- ✅ Server-side Form Validation
- ⚠️ Error Handling
- 📱 Responsive User Interface

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
- JavaScript
- HTML
- CSS

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/mayur-thakre-7526/wanderlust-project.git
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
node app.js
```
